# 第一部分：基础外设通信总线

<span class="badge-b">[B]</span> <span class="badge-i">[I]</span>


> **难度等级**：B → I
> 
> 本部分覆盖嵌入式系统中最常用的低速外设总线，> 从 I2C、SPI 到 UART、1-Wire，> 是每个嵌入式工程师必须掌握的基础技能。

---

## 本部分大章概览

| 大章 | 难度 | 核心内容 | 典型场景 |
| --- | --- | --- | --- |
| I2C | B → I | 两线串行、开漏驱动、地址仲裁 | 传感器、EEPROM、RTC |
| MIPI-I3C | I → E | I2C 升级版、动态地址、HDR 模式 | 手机传感器阵列、汽车 ECU |
| SPI | B → I | 四线全双工、片选驱动、时钟极性 | Flash、ADC、DAC、显示屏 |
| UART | B | 异步串口、波特率配置、流控 | 调试串口、GPS、蓝牙模块 |
| 1-Wire | B | 单线供电+通信、ROM ID 识别 | 温度传感器、门禁系统 |
| MDIO | I | 以太网 PHY 管理、MII 寄存器读写 | 交换机、路由器、工业以太网 |

---

## 学习路径建议

**路径 A（传感器方向）**：
I2C → MIPI-I3C → SPI → MDIO

**路径 B（通信方向）**：
UART → SPI → I2C → MDIO

**路径 C（快速入门）**：
UART → I2C → SPI（覆盖 90% 的嵌入式项目）

---

## 选型速查表

| 场景 | 推荐总线 | 原因 |
| --- | --- | --- |
| 连接温度传感器 | I2C | 两线、多设备、标准协议 |
| 高速 Flash 存储 | SPI | 全双工、50MHz+ 时钟 |
| 调试日志输出 | UART | 异步、无需额外时钟线 |
| 多传感器阵列（手机） | MIPI-I3C | 动态地址、中断聚合、低功耗 |
| 读以太网 PHY 状态 | MDIO | 标准化寄存器、热插拔检测 |
| 门禁卡/温度探头 | 1-Wire | 单线供电、唯一 ID、远距离 |
