## 分级学习路径
### 小白路径（B→I）
只学01→02→03基础部分→04→05，跳过所有M级标记内容。<br>
用最小Demo跑通流程，理解"是什么、怎么跑、为什么在嵌入式上这么做"。

### 高手路径（I→E）
完整学习01–07全部内容，重点吃透03核心原理、05-07全场景实战。<br>
能独立完成项目选型、部署、调优，解决量产稳定性问题。

### 百科全书价值（E→M）
覆盖原理→实战→演进→未来全链路，形成边缘AI推理在嵌入式Linux上的完整知识闭环。<br>
可用于架构设计、方案选型、问题根治、前沿预研，成为领域权威参考。
