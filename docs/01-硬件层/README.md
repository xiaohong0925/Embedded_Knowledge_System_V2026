# 01-硬件层

<span class="badge-b">[B]</span>

---

<span class="red">硬件层</span> 是嵌入式 Linux 知识体系的基础。理解处理器架构、内存映射、总线协议和外设接口，是进行内核开发和驱动编程的必要前提。

---

## <strong>模块概览</strong>

本模块涵盖嵌入式系统的硬件基础知识，包括：

| 章节 | 主题 | 难度 |
|------|------|------|
| 处理器架构 | ARM/RISC-V 体系结构、内存映射、Cache 原理 | [B→I] |
| 硬件接口 | I2C/SPI/UART/CAN、传感器、执行器 | [B→I] |
| 存储设备 | NAND/NOR/eMMC、文件系统适配、可靠性 | [I→E] |
| 电源管理 | DVFS、休眠唤醒、低功耗设计 | [I→E] |
