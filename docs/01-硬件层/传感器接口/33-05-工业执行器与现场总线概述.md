# 工业执行器与现场总线概述

> 📊 **本节难度等级：** <span class="badge-i">**I级**</span>

---

### <strong>`原理`：PWM原理：占空比、频率与等效电压/功率的关系。</strong>


### <strong>`命令`：通过SysFS接口配置和控制PWM，驱动一个LED调光或舵机。</strong>


### <strong>`讲解`：GPIO作为数字输出控制继电器/电磁阀。</strong>


### <strong>`讲解`：为什么工业领域需要现场总线？对比RS-485, CAN, CANopen, EtherCAT。</strong>


### <strong>`原理`：核心差异：通信速率、拓扑结构、同步精度、软硬件要求。</strong>


### <strong>`场景`：CAN在汽车车身控制，EtherCAT在工业机器人/高端CNC的应用。</strong>
