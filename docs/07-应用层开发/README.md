# 应用层开发 [B→E]

> **本章学习目标**：
> - 掌握嵌入式 Linux 多线程编程、进程间通信、网络编程核心技术
> - 理解轻量级容器化技术在嵌入式场景中的裁剪与部署
> - 掌握低功耗设计、性能分析、系统服务化的工程实践

---

## 本章结构

| 章节 | 主题 | 徽章 | 核心内容 |
|------|------|------|---------|
| **多线程编程** | 线程基础、同步、实时 | [B]→[I] | pthread、原子操作、死锁防治、PREEMPT_RT |
| **进程间通信** | IPC 机制与选型 | [I] | 管道、消息队列、D-Bus、共享内存、RPMsg |
| **网络编程** | Socket 与协议栈 | [I] | TCP/UDP、epoll、lwIP、MQTT/CoAP |
| **开发工具链** | 调试与构建工具 | [B] | GDB、交叉编译、CMake、构建系统 |
| **轻量级容器化技术** | 嵌入式容器化 | [I]→[E] | Docker 裁剪、Buildroot 容器、systemd-nspawn |
| **低功耗与可靠性设计** | 功耗与稳定性 | [I] | 看门狗、DVFS、休眠策略、Runtime PM |
| **性能分析与优化** | Profiling 工具链 | [I]→[E] | strace、perf、火焰图、Valgrind |
| **系统服务化** | systemd 与守护进程 | [I] | unit 管理、daemonize、日志轮转、故障恢复 |

---

## 学习路径建议

- **初级（B）**：多线程编程前 3 节 → 开发工具链 → 网络编程基础
- **中级（I）**：线程同步/死锁/实时 → IPC 全系列 → 容器化 → 系统服务化
- **高级（E）**：性能分析（perf/Valgrind）→ 低功耗深度优化 → 容器化 OTA 方案

---

## 前置知识

- C/C++ 编程基础
- Linux 系统调用与文件 IO
- Makefile / CMake 构建基础
