# 基础外设通信总线

<span class="badge-b">[Beginner]</span>

<span class="red">基础外设通信总线</span> 是嵌入式Linux系统中 低速外设与传感器接口 的重要组成部分。

---

## <strong>模块概览</strong>

<span class="green">基础外设通信总线</span> 涵盖以下总线类型：

- <span class="green">I2C</span> — 两线串行总线，多主多从<br>
- <span class="green">SPI</span> — 四线高速同步串行<br>
- <span class="green">UART</span> — 异步串行通信，调试必备<br>
- <span class="green">1-Wire</span> — 单线极简总线，温度传感器<br>
- <span class="green">MDIO</span> — 以太网PHY管理接口<br>
- <span class="green">MIPI-I3C</span> — I2C升级版，更高速度

## <strong>总线选型速查表</strong>

| 总线 | 速率 | 距离 | 拓扑 | 适用场景 |
|------|------|------|------|----------|
| I2C | 400k-3.4M | <1m | 多主多从 | 传感器、EEPROM、RTC |
| SPI | 10-100M | <0.5m | 一主多从 | 显示屏、Flash、ADC |
| UART | 115k-4M | <15m | 点对点 | 调试串口、GPS、蓝牙 |
| 1-Wire | 16k | <100m | 单主多从 | 温度传感器、iButton |
| MDIO | 2.5M | <1m | 一主多从 | 以太网PHY寄存器配置 |
| MIPI-I3C | 12.5M | <1m | 多主多从 | 移动设备传感器、摄像头 |
