# 调试与跟踪专用总线

<span class="badge-b">[Beginner]</span>

<span class="red">调试与跟踪专用总线</span> 是嵌入式Linux系统中 芯片调试、跟踪与烧录 的重要组成部分。

---

## <strong>模块概览</strong>

<span class="green">调试与跟踪专用总线</span> 涵盖以下总线类型：

- <span class="green">CoreSight/ETM</span> — ARM调试架构<br>
- <span class="green">JTAG</span> — 边界扫描与调试<br>
- <span class="green">SWD</span> — 串行线调试，2线替代

## <strong>总线选型速查表</strong>

| 总线 | 速率 | 距离 | 拓扑 | 适用场景 |
|------|------|------|------|----------|
| CoreSight | 可变 | 片上 | 星型 | ARM SoC调试、跟踪 |
| JTAG | 10-100M | <0.3m | 菊花链 | 边界扫描、芯片烧录 |
| SWD | 10-100M | <0.3m | 点对点 | Cortex-M调试、低成本 |

## <strong>小结与练习</strong>

| 要点 | 说明 |

| 核心概念 | 调试与跟踪专用总线 的协议分层与选型原则 |
