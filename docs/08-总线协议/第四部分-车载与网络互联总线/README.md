# 第四部分：车载与网络互联总线

<span class="badge-i">[I]</span> <span class="badge-e">[E]</span>


> **难度等级**：I → E
> 
> 本部分覆盖<span class="red">汽车电子和工业网络</span>中使用的总线协议。
> 
> 从 CAN 的差分仲裁到 TSN 的确定性调度，从 LIN 的低成本车身控制到车载以太网的多媒体传输，
> 这些协议定义了现代车辆的"神经系统"。

---

## 本部分大章概览

| 大章 | 难度 | 核心内容 | 典型场景 |
| --- | --- | --- | --- |
| CAN | I → E | CSMA/CD+仲裁、CAN FD、ISO-TP | 车身控制、动力系统 |
| LIN | B → I | 单主多从、低成本、UART衍生 | 车门、座椅、灯光 |
| 车载以太网 | I → E | 100BASE-T1、AVB/TSN、DoIP | ADAS、信息娱乐 |
| TSN | E → M | IEEE 802.1Qbv、时间同步、门控调度 | 自动驾驶、工业控制 |

---

## 学习路径建议

**路径 A（车身电子工程师）**：
LIN → CAN → CAN FD

**路径 B（ADAS/自动驾驶）**：
CAN FD → 车载以太网 → TSN

**路径 C（全栈车载）**：
LIN → CAN → 车载以太网 → TSN

---

## 选型速查表

| 场景 | 推荐总线 | 原因 |
| --- | --- | --- |
| 车门/座椅/灯光 | LIN | 成本最低，单线布线 |
| 发动机/变速箱 | CAN | 实时性，高可靠 |
| 高清摄像头 | 车载以太网 100BASE-T1 | 带宽 100Mbps，单对线 |
| 激光雷达数据 | 车载以太网 1000BASE-T1 | 带宽 1Gbps |
| 传感器时间同步 | TSN (gPTP) | ns 级同步精度 |
