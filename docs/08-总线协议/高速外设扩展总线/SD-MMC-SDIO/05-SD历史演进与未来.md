# SD历史演进与未来

<span class="badge-i">[Intermediate]</span>

<span class="red">SD（Secure Digital）卡</span> 是最广泛使用的可移动存储标准，从数码相机扩展到物联网全领域。

---

## <strong>历史演进</strong>

- <span class="green">1999 年 SD 1.0</span> — 基于MMC，加入安全机制，最高 12.5MB/s<br>
- <span class="green">2003 年 SD 1.1</span> — 引入SDIO，支持WiFi/蓝牙模块<br>
- <span class="green">2006 年 SDHC</span> — 2GB-32GB，FAT32文件系统<br>
- <span class="green">2009 年 SDXC</span> — 32GB-2TB，exFAT文件系统，UHS-I 104MB/s<br>
- <span class="green">2013 年 UHS-II</span> — 第二排针脚，156MB/s<br>
- <span class="green">2017 年 UHS-III</span> — 624MB/s<br>
- <span class="green">2018 年 SD Express</span> — 融合PCIe，985MB/s-1969MB/s<br>
- <span class="green">2024 年 SD 9.0</span> — 支持CXL.mem扩展

## <strong>eMMC与UFS替代关系</strong>

| 接口 | 速度 | 容量 | 适用 |
|------|------|------|------|
| SD/microSD | 104MB/s-2GB/s | 1TB+ | 可移动、消费级 |
| eMMC | 400MB/s | 128GB | 低成本板载 |
| UFS | 2.3GB/s | 1TB+ | 高端手机、汽车 |

<span class="blue">嵌入式系统根据是否需要热插拔在SD和eMMC/UFS之间选择</span>。
