# 高速外设扩展总线

<span class="badge-b">[Beginner]</span>

<span class="red">高速外设扩展总线</span> 是嵌入式Linux系统中 高速数据传输与外设扩展 的重要组成部分。

---

## <strong>模块概览</strong>

<span class="green">高速外设扩展总线</span> 涵盖以下总线类型：

- <span class="green">PCIe</span> — 高速串行扩展总线，PC/服务器标准<br>
- <span class="green">SATA</span> — 串行ATA，存储设备接口<br>
- <span class="green">SD/MMC/SDIO</span> — 可移动存储与IO扩展<br>
- <span class="green">USB</span> — 通用串行总线，万能接口

## <strong>总线选型速查表</strong>

| 总线 | 速率 | 距离 | 拓扑 | 适用场景 |
|------|------|------|------|----------|
| PCIe | 8-128GT/s | <0.3m | 点对点/交换 | 显卡、NVMe SSD、网卡 |
| SATA | 6Gbps | <1m | 点对点 | 硬盘、SSD、光驱 |
| SD/MMC | 104M-2GB/s | <0.1m | 点对点 | 可移动存储、WiFi/BT模块 |
| USB | 480M-80Gbps | <5m | 星型/Hub | 外设、存储、网络、调试 |
