## 终点成就检查清单

完成第一部后，你应该能独立完成以下任务：

1. [ ] 拿到一块新开发板，能识别其SoC、存储介质和调试接口
2. [ ] 成功连接串口，看到BootROM/U-Boot的输出
3. [ ] 配置并编译交叉编译工具链，或正确使用预构建工具链
4. [ ] 编译U-Boot并烧录到开发板，能进入U-Boot命令行
5. [ ] 配置、编译Linux内核，生成支持目标板的Image
6. [ ] 使用TFTP/NFS建立网络开发环境
7. [ ] 手工构建BusyBox根文件系统，编写init脚本
8. [ ] 看到shell提示符，执行基本命令（ls、cat、ifconfig等）
9. [ ] 从用户空间控制GPIO，点亮/熄灭LED
10. [ ] 编写设备树节点，加载LED驱动，通过sysfs控制LED

- "U-Boot怎么知道板子上有多少内存？" → 第II部第7章：设备树深入与内存管理子系统

- "每次手工编译太麻烦，有自动化的方法吗？" → 第II部第6章：Buildroot与Yocto构建系统
- "根文件系统只有BusyBox太简陋了，怎么添加更多功能？" → 第II部第9章：系统服务与进程管理
- "内核启动时打印的那些日志到底是什么意思？" → 第II部第7章：内核子系统深入
- "GPIO只有sysfs一种控制方式吗？" → 第II部第8章：字符设备接口与ioctl
