## 知识点50：本章核心概念回顾 [B] ~700字

如果你把本章比作一次"拆箱组装"之旅，那么你现在应该已经完整走过了以下四站：认识零件→连接线路→按下开关→发出第一道指令。

### 第一站：开发板的"五脏六腑"

任何嵌入式开发板本质上都是五大件的乐高组合：

| 组件 | 日常比喻 | 本章对应内容 |
|------|----------|-------------|
| SoC（处理器） | 大脑 | 芯片型号、ARM架构 |
| DDR（内存） | 临时工作台 | 容量大小、运行内存 |
| 存储介质 | 文件柜 | eMMC / SD卡 / SPI Flash |
| 电源管理 | 心脏+血管 | DC接口、稳压芯片 |
| 外设接口 | 手脚眼耳 | 串口、网口、USB、GPIO |

💡 **提示**：记不住时，记住这个公式即可：**SoC + DDR + 存储 + 电源 + 接口 = 一块能跑Linux的最小系统**。

### 第二站：三步连接法

从拿到板子到看到开机信息，标准动作只有三步：

1. **接电源** — 确认电压（常见5V/12V）和电流是否匹配
2. **接串口** — USB转串口线连好，PC端打开串口工具（如Mobaxterm、minicom）
3. **按下电源键** — 观察串口是否有文字输出

⚠️ **陷阱**：很多初学者接好线却看不到输出，90%的原因是串口参数不对——必须是 **115200 8-N-1**（波特率115200，8位数据位，无校验，1位停止位）。

### 第三站：启动流程初识

上电后，开发板内部按固定剧本执行：

```mermaid
graph LR
    A[上电] --> B[BootROM<br/>芯片内固件]
    B --> C[加载U-Boot<br/>到DDR]
    C --> D[U-Boot启动<br/>串口输出信息]
    D --> E[等待用户命令<br/>或自动引导系统]
    style A fill:#e1f5e1
    style D fill:#fff4e1
```

[图1：嵌入式设备从上电到U-Boot的启动链路]

BootROM是芯片出厂时烧死的一段代码，用户改不了，它的唯一任务是找到U-Boot并把它加载到内存。U-Boot被加载后，你才能在串口终端上看到熟悉的启动日志和提示符 `=>`。

### 第四站：你在U-Boot下学到的四条命令

本章实操环节，你在U-Boot命令行里实际敲过的命令如下：

```bash
=> help          # 查看所有可用命令
=> version       # 查看U-Boot版本号
=> printenv       # 打印所有环境变量
=> bdinfo         # 查看板级信息（内存大小、MAC地址等）
```

这四条命令的共同特点是**只读、安全**，随便敲都不会破坏系统，非常适合新手练手。

🔴 **危险**：本章你还没有学习 `saveenv`、`erase`、`nand write` 等写操作命令——**它们能改配置甚至擦除固件，第2章之前不要碰**。

## 本节总结

下表是本节乃至本章的核心回顾总表。建议保存或打印，随时查阅：

| 序号 | 知识板块 | 核心要点 | 关键操作/数值 | 常见错误 |
|:----:|----------|----------|---------------|----------|
| 1 | 开发板组成 | SoC+DDR+存储+电源+接口 | 认准主芯片型号 | 把GPIO口当电源接 |
| 2 | 供电连接 | 确认电压、电流、极性 | 5V/12V直流 | 反接烧毁稳压芯片 |
| 3 | 串口连接 | USB转TTL串口线 | TX/RX交叉连接 | TX-TX直连导致无输出 |
| 4 | 串口参数 | 波特率+数据位+校验+停止位 | **115200 8-N-1** | 参数不对全黑屏 |
| 5 | 上电启动 | 电源键或自动上电 | 观察串口是否有输出 | 没装驱动识别不到串口 |
| 6 | BootROM | 芯片内置、只读 | 自动加载第一阶段引导 | 误以为可以修改 |
| 7 | U-Boot | 开源引导加载程序 | 显示 `=>` 提示符 | 与操作系统混淆 |
| 8 | help命令 | 查看命令列表 | `help` 或 `?` | 无 |
| 9 | version命令 | 确认U-Boot版本 | `version` | 无 |
| 10 | printenv命令 | 查看环境变量 | `printenv` | 误以为是Linux的env |
| 11 | bdinfo命令 | 查看板级硬件信息 | `bdinfo` | 无 |
