## 三、本章核心公式速查

| 场景 | 公式/规则 |
|------|-----------|
| 页缓存回写触发阈值 | `dirty_background_ratio` (默认 10%) < 脏页比例 < `dirty_ratio` (默认 20%) |
| NAND 实际可用容量 | `用户容量 = (总PEB数 - 坏块 - UBI头 - 预留) × (PEB大小 - 2页)` |
| UBI 最小开销 | 每 PEB 保留 2 页（首/尾）存 EC 头与 VID 头 |
| F2FS GC 效率 | `有效页比例 < 阈值` 时触发 GC；预留空间越大，GC 频率越低 |
| ext4 日志事务大小 | 受 `journal_size` 限制，默认按块组数自动计算 |
| barrier 语义 | `WRITE → FLUSH → FUA` 保证之前写入全部落盘后才返回 |
